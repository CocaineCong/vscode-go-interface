# vscode-go-interface 

Go语言interface跳转的vscode插件

## 功能

- 支持Go语言的interface跳转

![interface](./images/interface.png)

- 支持Go语言的interface实现的函数跳转

![interface_nav](./images/interface_nav.png)

![implementation](./images/implementation.png)
